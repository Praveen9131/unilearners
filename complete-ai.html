Course Plan

Weeks 1-2: Introduction to AI

  • Overview of AI: History, applications, and future trends
  • Setting up Python for AI development
  • Project: Build a simple chatbot using basic NLP

Weeks 3-4: Foundations of Machine Learning

  • Supervised vs. unsupervised learning
  • Regression, classification, and clustering algorithms
  • Hands-on with scikit-learn: Build a spam email classifier

Weeks 5-6: Data Preprocessing and Feature Engineering

  • Data cleaning, normalization, and encoding
  • Feature selection and dimensionality reduction (PCA)
  • Project: Prepare a dataset for a machine learning model

Weeks 7-8: Deep Learning Fundamentals

  • Introduction to neural networks and TensorFlow/Keras
  • Building and training a simple neural network
  • Project: Create an image classifier using CNNs

Weeks 9-10: Advanced Deep Learning

  • Convolutional Neural Networks (CNNs) for computer vision
  • Recurrent Neural Networks (RNNs) for time series
  • Project: Develop a handwritten digit recognition system

Weeks 11-12: Natural Language Processing (NLP)

  • Text preprocessing, tokenization, and embeddings
  • Building NLP models with Hugging Face Transformers
  • Project: Create a sentiment analysis tool

Weeks 13-14: Generative AI Basics

  • Introduction to GANs, VAEs, and diffusion models
  • Text-to-image generation with Stable Diffusion
  • Project: Generate artwork using diffusion models

Weeks 15-16: Advanced Generative AI

  • Fine-tuning large language models (LLMs) with Hugging Face
  • Retrieval-Augmented Generation (RAG) with LangChain
  • Project: Build a knowledge-based Q&A system

Weeks 17-18: Ethics and Responsible AI

  • Addressing bias, fairness, and transparency in AI
  • Case studies on ethical AI deployment
  • Project: Analyze and mitigate bias in a dataset

Weeks 19-20: Capstone Projects and Deployment

  • Develop a full AI application (e.g., AI-powered chatbot or image generator)
  • Deploy models using Flask and AWS/GCP
  • Present your portfolio to industry experts
